# Color2Value
<img width="2000" height="1414" alt="Sea Surface Salinity" src="https://github.com/user-attachments/assets/6eb60253-bbda-4fb2-b5b2-2cfaedb15101" />

A screen-reading digitizer to extract exact values from color maps.

First release of Color2Value! 
A desktop app that directly reads values from color maps and contour plots on your screen.

[Installation & Usage]

・Download Color2Value.zip from the Assets section below.
・Right-click the ZIP file and select "Extract All" (Unzip).
・Open the extracted folder and double-click Color2Value.exe to run the app in the background.
・Open any color map/plot on your screen and press Ctrl + F9 to start capturing!

※Important Note: Do NOT move Color2Value.exe out of its folder. Keep all files together in the extracted folder (You can create a desktop shortcut instead).
Note: On the very first run, it may take a few moments to initialize the OCR AI model.

Color2Value v1.0.0 を公開しました！ 
画面上のカラーマップやコンター図（水温図など）から、直接数値を読み取るデスクトップアプリです。

【インストールと使い方の手順】
・下の Assets から Color2Value.zip をダウンロードします。
・ダウンロードしたZIPファイルを右クリックして 「すべて展開（解凍）」 します。
・展開されたフォルダの中にある Color2Value.exe をダブルクリックすると、アプリがバックグラウンドで起動します。
・読みたい図を画面に表示し、Ctrl + F9 を押すと操作モードになります！

※注意: Color2Value.exe というファイル単体をデスクトップなどに移動させないでください。必ず解凍したフォルダに入れたまま実行（または右クリックでショートカットを作成）してください。
※一番最初の起動時のみ、文字認識AIの準備に少し時間がかかる場合があります。
